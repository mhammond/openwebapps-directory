This site is a reference implementation of a application directory for the Mozilla Open Web Apps project. We hope to see other developers outside of our group implementing directories and stores around this concept; this site is meant to serve as an example, but also as a way for people developing open web applications to show their work to each other, and to help the Mozilla team understand exactly what people are creating.

The code itself is located in a github project. If you encounter issues or have ideas for enhancements for this site the github issue tracker is a good place to keep those ideas. But note that this site not intended to be a complete example of a user-facing application directory. Mostly this is because we aren't the best people to write such a thing.

If you do write a directory or store, we want to hear about your experience! Please write to us on the Mozilla Labs Group. You can also find us on the #openwebapps IRC channel on irc.mozilla.org.
